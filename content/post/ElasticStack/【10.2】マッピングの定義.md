### 要点
- 独自のマッピングの定義方法
  - インデックス`mappings`セクションに定義


---
### 内容
- インデックス作成時にマッピングを定義

```
PUT my_index
{
 "mappings": {
 define mappings here
 }
}
```

- 既存インデックスにマッピングを追加

```
PUT my_index/_mapping
{
 additional mappings here
}
```

- ドキュメントをインデックスすると、ESは動的にマッピングを作成、更新する
  - マッピングが存在しない場合、新しく作成
  - マッピングでまだ定義されていないフィールドを追加
