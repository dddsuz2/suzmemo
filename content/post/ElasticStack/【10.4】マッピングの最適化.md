### 要点
- よりデータの範囲に特化した型にマッピングを最適化
  - メモリ削減、性能の最適化


---
### 内容
- マッピングは変更可能？
  - ドキュメントの再インデックスなしではできない
    - フィールドのデータ型を変更してしまうと、変更前にすでにインデックスしていた値は検索できなくなる
  - 新しいフィールドの追加は可能

- マッピングの修正
```
PUT blogs_v2
{
	"mappings": {
		"properties": {
			"publish_date": {
				"type": "date"
			}
		}
	}
}
```
