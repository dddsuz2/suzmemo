- スライスによる要素の書き換え
  - スライスを使ってサブリストに値を代入することができる

```python
>>> numbers = [1, 2, 3, 4]
>>> numbers[1:3] = [8, 9]
>>> numbers
[1, 8, 9, 4]
```

- リストに代入しようとしている右辺値は、左辺のスライスと要素数が等しくなくてもよい

```python
>>> numbers = [1, 2, 3, 4]
>>> numbers[1:3] = [7, 8, 9]
>>> numbers
[1, 7, 8, 9, 4]
>>> numbers[1:3] = []
>>> numbers
[1, 9, 4]
```

- 右辺値はリストでなくてもよい
  - Pythonのイテラブルなら何でもよい

```python
>>> numbers = [1, 2, 3, 4]
>>> numbers[1:3] = (98, 99, 100)
>>> numbers
[1, 98, 99, 100, 4]
>>> numbers = [1, 2, 3, 4]
>>> numbers[1:3] = 'wat?'
>>> numbers
[1, 'w', 'a', 't', '?', 4]
```
