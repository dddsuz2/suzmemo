- 1つのリストを複数の変数に代入すると、その中の1つでリストを書き換えたときに他も書き換えられる
  - 2つのタグについている糸はどちらも同じ箱につながっている
```python
>>> a = [1, 2, 3]
>>> a
[1, 2, 3]
>>> b = a
>>> b
[1, 2, 3]
>>> a[0] = 'suprise'
>>> a
['suprise', 2, 3]
>>> b
['suprise', 2, 3]
```
