- 新しいオブジェクトを作るときには、そのオブジェクトの内容を規定するクラスを作らなければならない
  - オブジェクトを箱とするなら、クラスは鋳型のようなもの

```python
>>> class Cat:
...     pass
```

- オブジェクトは、関数と同じようにクラス名を呼び出して作る

```python
>>> a_cat = Cat()
>>> another_cat = Cat()
```
