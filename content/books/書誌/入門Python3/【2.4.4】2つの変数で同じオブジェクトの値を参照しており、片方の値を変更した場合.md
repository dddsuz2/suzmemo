- イミュータブルなオブジェクトの場合
```python
>>> x = 5
>>> y = x
>>> x = 29
>>> x
29
>>> y
5
```

- ミュータブルなオブジェクトの場合
```python
>>> x
[1, 2, 3, 4]
>>> y = x
>>> y
[1, 2, 3, 4]
>>> x
[1, 2, 3, 4]
>>> x.append(5)
>>> x
[1, 2, 3, 4, 5]
>>> y
[1, 2, 3, 4, 5]
```

リストの場合、x[0], x[1], x[2] 自体も名前のような機能をはたしている
- それぞれが、1, 2, 3 を指している
