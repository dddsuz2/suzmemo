### 内容

| パターン | マッチ                                            | 
| -------- | ------------------------------------------------- | 
| \d       | 1個の数字                                         | 
| \D       | 1個の数字以外の文字                               | 
| \w       | 1個のUnicode単語文字（または英数字と_）           | 
| \W       | 1個のUnicode単語文字（または英数字と_）以外の文字 | 
| \s       | 1個の空白文字                                     | 
| \S       | 1個の空白以外の文字                               | 
| \b       | 単語の境界                                        | 
| \B       | 単語の境界以外の文字間                            |
