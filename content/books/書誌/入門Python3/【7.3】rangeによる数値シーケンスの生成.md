`range()`関数を使うことで、指定した範囲の数値のシーケンスを返すことができる
- あらかじめリストやタプルなどの大きなデータ構造を作って、そこに値を格納する必要もない
- `range(start, stop, step)`のように、スライスとよく似た形式で使う
  - 指定しない場合、`start`はゼロから
  - `stop`は唯一必須
  - `-1`を指定することで、`stop`を逆順にできる
- `zip()`と同様に、`range()`はイテラブルなオブジェクトを返す
  - `for ... in`で反復処理する
  - リストなどのシーケンスに変換する

```python
>>> for x in range (0, 3):
...     print(x)
...
0
1
2
>>> list(range(0, 3))
[0, 1, 2]
>>> for x in range(2, -1, -1):
...     print(x)
...
2
1
0
>>> list(range(2, -1, -1))
[2, 1, 0]
>>> list(range(0, 11, 2))
[0, 2, 4, 6, 8, 10]
```
