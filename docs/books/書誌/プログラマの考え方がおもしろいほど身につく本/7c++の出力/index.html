問題：カウントダウンとカウントアップ
以下のコードのループ内に書く式（expression）を考えてみよう。このプログラムは、5 から 1 まで
の数を一行に 1 つずつ順番に表示するものである。
for (int row = 1; row <= 5; row++) {
cout << 1 expression << "\n";
}
  • expressionにどのような値が入ればよいか？
    • rowに-1をかけた値はどうだろう？
      • 数は減るが、欲しい値ではない
    • 欲しい値と、行番号-1 との差は常に6
    • 6-rowにすればよくね？
for (int row = 1; row <= 5; row++) {
    for (int hashNum = 1; hashNum <= 6 - row; hashNum++) {
        cout << "#";

using namespace std;

int main()
{
    for (int i = 1; i < 8; i++) {
        for (int j = 1; j <= 4 - abs(4 - i); j++ ) {
            cout << "#";
        }
        cout << "\n";
